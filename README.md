# AngularStrapi app

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 7.3.9. and [Strapi](https://github.com/strapi/strapi) version 3.0.0.

## Development server

Run `docker-compose -d up` for a dev server.
Navigate to `http://localhost:4200/`. The app will automatically reload if you change any of the source files.
Navigate to `http://localhost:1337/admin`, configure your Strapi backend.


## Build

Run `docker-compose up --build` to build your docker image.
